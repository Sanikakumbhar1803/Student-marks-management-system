# Student Marks Management System - SAP ABAP on HANA

## Description
Developed custom backend components using SAP ABAP Development for HANA to design, code, and enhance student data processing modules. Created database tables and internal table operations to deliver scalable, high-performing report functionality. Built following agile practices with focus on clean code and performance optimization.

## Key Features
- Database table creation using SE11 - ZSTUDENT table with fields RollNo, Name, Marks
- ABAP Report Program Z_STUDENT_REPORT using SELECT queries on HANA database
- Internal table processing for sorting and filtering student records
- Optimized backend logic for fast data retrieval

## Technical Stack
- Language: SAP ABAP
- Database: SAP HANA 
- Tools: SE11, SE80, SE16N
- Concepts: Data Dictionary, Internal Tables, Open SQL

## How to Run
1. Create table ZSTUDENT in SE11
2. Execute report Z_STUDENT_REPORT in SE38
3. Enter selection criteria and view output
